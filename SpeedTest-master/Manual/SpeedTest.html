On the Subject of Speed Test

No, this is not the chart. This is patrick! Come get your order as quick as you can!

On the module, there is a large button, which is the only thing you can press on it. The button cycles through 10 colors, from the list: Red, Green, Blue, Yellow, Cyan, Magenta, White, and Black, in this exact order.

How to solve the module.

The first step is to calculate the number of presses, which can be calculated by taking each color (excluding the orange), getting each R, G, and B value as a sum, and using this formula.

Sum of R values * Sum of G values * Sum of B values.

However, there is a time limit that starts when you start the first press. Just take the number of presses and divide by 5 to get it. The time limit is always in seconds.

If you get at least the number of required presses before the time limit is reached, the module will solve. But if you don't get enough presses when the time limit is reached, you will strike.
